# SweetShop
The SweetShop project is a Spring Boot application designed to manage a sweet shop. It uses Java 21, Spring Boot, Spring Data JPA, MySQL, and optionally Spring Security for authentication.
## Project Overview

SweetShop is designed to help a sweet shop manage its inventory and orders easily.  
The application has the following functionalities:

- Browse available sweets and categories
- User authentication and login
- Add items to the cart
- Place orders
- Admin panel to manage inventory (optional)

## Tech Stack

- **Backend:** Java, Spring Boot, Spring Data JPA
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Build Tool:** Maven
- **Server:** Apache Tomcat (embedded)
